# LSBuilder

Diese Seite wird mit **MkDocs** veröffentlicht.

## Über dieses Projekt

LSBuilder ist ein Repository zur Erstellung von **SchuCu-BBS-konformen Lernsituationen** (berufsbildende Schulen Niedersachsen) auf Basis der Leitlinie **„Schulisches Curriculum berufsbildende Schulen (SchuCu-BBS) 2024/07"**.

## Verfügbare Lernsituationen

### Lernfeld 1: Das Unternehmen und die eigene Rolle im Betrieb beschreiben
- [LS 1.1: Unternehmensformen verstehen und auswählen](lernsituationen/LS_01_unternehmensformen/index.md)

[Alle Lernsituationen anzeigen](lernsituationen/index.md)

## Hinweise

- Inhalte liegen unter `docs/`
- Deployment erfolgt via GitHub Actions auf den Branch `gh-pages`
- Alle Lernsituationen sind dreistufig differenziert (Niveau A/B/C)
- Pro Niveau steht ein Moodle-kompatibles Quiz (GIFT-Format) zur Verfügung
